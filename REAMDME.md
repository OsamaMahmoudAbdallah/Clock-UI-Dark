# JavaScript Clock UI (Dark)

A live analog clock built with vanilla HTML, CSS, and JavaScript — dark themed, with smooth rotating hour, minute, and second hands.

## Features
- Real-time analog clock synced to system time
- Smooth hand rotation using CSS transforms, updated via `setInterval`
- Dark, minimal clock face design

## Tech Stack
- HTML5
- CSS3 (flexbox, transforms, box-shadow for depth)
- Vanilla JavaScript (Date API, DOM manipulation)

## How to Use
1. Open `index.html` in any browser
2. The clock starts running immediately — no setup needed

## Project Structure
- `index.html` — markup and clock logic
- `style.css` — dark theme styling and hand positioning
- `clock.png` — clock face background image (numbers/ticks)
